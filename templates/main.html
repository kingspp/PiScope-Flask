Instructions for Pi Generation

Modes

1. Ethernet Mode

a. In ethernet mode, the raspberry pi is connected to a LAN network via a Router
b. Pi is assigned a static IP "192.168.0.99" (For Router Only)
c. Power on Raspberry Pi.
d. Open your browser and type 192.168.0.99, press enter.
e. You should see the PiScope website, if not, check the steps or contact the Developer.

2. HotSpot Mode

a. In HotSpot mode, the raspberry pi is connected to a Mobile device via the device's HotSpot Configuration
b. Wifi Hotspot Configuration: (Create using the following credentials)
*HotSpot SSID (name): PiScope
*HotSpot Key_Mgmt : WPA (WPA1)
*HotSpot password : 9738134205
c. Pi is assigned to a static IP "192.168.43.51" (*Usually)
d. Power on Raspberry Pi.
e. Open your browser and type IP, press enter.
f. You should see the PiScope website, if not continue the guide.
g. Download WiFi hotspot Manager from Playstore.
h. In clients tab , copy the IP Address and repeat from step e
